# 학습 시드 — 확정 원칙 (generate 주입)
> AGENT_LEARNINGS_HANDOFF distill. 이 파일은 generate.mjs가 프롬프트에 주입한다. 피드백 distill로 갱신/추가된다.

## 콘텐츠
- 캐러셀 = 하나의 thesis + 한 방향 스토리. 사실 나열 금지. 처음 연 프레임으로 끝까지(프레임 일관성).
- 과장·허위 전제 금지("원래 ~였다" 일반화 단정 X). 길이 7~10장. 4~5장은 빈약.
- 커버는 후킹(질문형·통념 반박·아포리즘 OK). 마무리는 자기완결 선언 한 줄(메타 안내 금지, 넛지로).
- 키워드 과반복 금지(검정·기준). 제목은 직관적·평이.

## 톤(레이어별)
- 선언/여정(#1~3): 약간의 무게 OK. 데이터(#4~7): 담백·정밀. 정체성·모집·제품(#8~10): 애플식 명료·긍정 초대(비장·과장·부정 함의 금지).
- 불필요한 쉼표 제거(평서문은 끊지 않는다). 피드백 반영은 최소 변경 원칙.

## 사실·용어 (절대)
- 검출/불검출(N.D.)=유해물질 자가품질검사 전용. ISO 10993-23=자극 지수/측정 범위 최솟값. 세포독성=생존율 97%(비독성 기준 70%).
- 시험은 공인기관 수행("직접/자체 측정" 금지). 반영구 깊이: "표피에 올리되 일부는 진피까지 남는다"(타투 혼동 금지).
- '수치'→'숫자'. 멜라누아=제조사("만든다"). 미게시 수치 금지(예: 0.5g→'일회분량').

## 비주얼
- 내부 슬라이드는 배경을 죽인다(A/B/C/D/E). 실사 컬러 유지(흑백 변환 금지). 텍스트 시작 위치 고정.
- 핵심 1문장만 흰색+골드밑줄 인라인(슬라이드당 1개). 회색 부연 2~3문장(60~110자).
- 표지 이미지 본문 비재사용. 이미지 캐러셀당 ≤2(2회째 변형). 임팩트 이미지 1회. 의료가운 금지.

## 교정 패턴(반복 금지)
- 질문↔답 동어반복 금지(질문에 답의 핵심어 미리 넣지 말 것).
- 부정 함의("~가 아니면 당당하지 않다") 금지 → 기능적 이득으로.
- 슬라이드에 마감일·링크·"→URL" 금지 → 캡션으로.
