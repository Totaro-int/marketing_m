> 엔진 폴더에 넣고 명령 한 줄이면 끝입니다. **다 한 뒤 이 메일은 꼭 삭제**해 주세요(보안).

---

## 1. 파일 넣기

첨부된 **`.env.local`** 을 엔진 폴더(`melanoir-studio`)의 **맨 위(루트)** 에 넣으세요:

→ `melanoir-studio/.env.local`

> **`.env.local` 은 점(`.`)으로 시작 = 숨김 파일**입니다. 맥 Finder에서 안 보이면 **`⌘ + Shift + .`** 로 표시를 켜세요.
> 메일 앱이 첨부 이름을 바꾸면(예: `env.local.txt`) — 받은 파일을 폴더에 두고 이름을 정확히 **`.env.local`** 로 바꿔주세요.

*(`web/config.js` 는 따로 받을 필요 없습니다 — 아래에서 `.env.local` 값으로 자동 생성됩니다.)*

---

## 2. 자동 생성 + 확인

엔진 폴더에서:

```bash
npm run setup        # (이미 설치했다면)  npm run gen:config
npm run doctor
```

→ `web/config.js` 가 `.env.local` 값으로 **자동 생성**되고, **✓ 준비 완료** 가 뜨면 끝입니다.
→ ✗ 가 있으면 Cowork에서 **"오류났어, RUNBOOK 보고 고쳐줘"** 라고 하세요.
