════════════════════════════════════════════════════════════

[사용법]
1) 함께 받은 첨부파일 ".env.local" 을 컴퓨터에 저장해 두세요. (열어볼 필요 없음)
2) Claude Code(또는 Claude Desktop, "코드 실행" 켜기)에 아래
   ┌─ 복사 시작 ─┐ ~ └─ 복사 끝 ─┘ 사이 내용을 "그대로 복사해서 붙여넣으세요".
3) 붙여넣을 때 그 .env.local 파일도 함께 첨부(또는 경로를 알려)주세요.
   나머지 설치·점검·오류 자가수정은 Claude가 알아서 합니다.

────────────────────────────────────────────────────────────
┌─ 복사 시작 ─┐

멜라누아 스튜디오를 설치해줘. 아래를 순서대로 하고, 오류가 나면
docs/RUNBOOK.md 를 읽고 그 지침대로 스스로 고쳐서 전부 ✓ 될 때까지 반복해.

1) 공개 저장소를 클론하고 그 폴더를 작업 폴더로 열어:
   git clone https://github.com/Totaro-int/marketing_m.git melanoir-studio

2) ".env.local" 파일을 melanoir-studio 폴더 맨 위(루트)에 놓아야 해.
   - 이 대화에 .env.local 을 첨부했으면 그 내용을 읽어서 melanoir-studio/.env.local 로 저장해.
   - 첨부가 없으면, 다운로드 폴더 등에서 .env.local 을 찾아 그 폴더로 복사해.
   - 못 찾으면 나(사용자)에게 "메일로 받은 .env.local 을 melanoir-studio 폴더에 넣어달라"고
     알려줘. 넣은 뒤 이어서 진행.
   (이 파일엔 Supabase 키가 들어있어 — 저장소에 커밋하거나 외부/공개 채널에 절대
    노출하지 마. 로컬 폴더에만 둬.)

3) melanoir-studio 폴더에서 설치와 점검을 실행해:
   npm run setup      (npm install + web/config.js 자동 생성 + 점검)
   npm run doctor     (준비 상태 점검 — 전부 ✓ 여야 함)

4) doctor 에 ✗ 가 있으면 docs/RUNBOOK.md 에서 그 [CODE]/증상을 찾아 지침대로
   고치고 npm run doctor 를 다시 실행해. 전부 ✓ 될 때까지 반복해.

5) 설치가 끝나면:
   - npm run doctor 결과를 보여주고
   - README.md 와 docs/INSTALL-CLIENT.md 의 "매일 사용" 부분을 짧게 요약해서 알려줘
   - "오늘 콘텐츠 만들어줘" 로 첫 콘텐츠 생성을 한 번 시연해줘
     (인스타·링크드인·스레드·네이버블로그·자사몰 인사이트 5채널 초안 + 검토 콘솔 안내)

참고: 매일 "/melanoir" 한 줄 명령으로 쓰고 싶으면 Claude Desktop → 설정 → Skills
에서 melanoir-studio/melanoir-skill.zip 을 업로드하면 돼(이건 화면 조작이라 사용자가
직접 해야 함). 업로드가 번거로우면 그냥 "오늘 콘텐츠 만들어줘" 라고 말해도 똑같이 동작해.

└─ 복사 끝 ─┘
