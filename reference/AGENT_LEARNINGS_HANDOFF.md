## A. 확정 선호 원칙 (에이전트가 따라야 할 규칙)

### A-1. 콘텐츠/스토리
1. **캐러셀 = 하나의 핵심 메시지(thesis) + 그걸 향해 한 방향으로 흐르는 스토리.** 사실 나열 금지.
2. **논리 일관성:** 처음 연 프레임으로 끝까지 간다. 한 프레임으로 열고 다른 프레임으로 닫지 않는다. (예: ‘무엇을 넣느냐’로 열고 ‘수치’로 닫으면 깨짐.)
3. **과장·허위 프레임 금지.** ("검정은 늘 넣어왔다" 같은 근거 없는 단정 X.)
4. **길이: 표지 포함 7~10장**(커버1+본문5~8+마무리1). 4~5장은 빈약.
5. **커버 후킹:** 무엇에 대한 내용인지 즉시 감 잡히게 + 궁금증. 숫자만 던지지 말 것.
6. **마무리:** 자기완결. "다음 글에서 보여드립니다" 류 메타 안내 금지 → **넛지(리프레임/질문)** 로 자연스러운 다음 행동 유도. **링크·상세는 슬라이드가 아니라 캡션(더보기란)** 에. "→ URL" 화살표 금지.
7. **용어 정확:** "반영구 화장/타투"(O) / "반영구 검정"(X) 등.

### A-2. 비주얼/레이아웃
1. **커버=실사 풀이미지(후킹)** / 데이터 커버=**가운데 큰 숫자(E)** + 짧은 회색 라벨.
2. **내부 슬라이드=배경을 죽인다**(텍스트가 주인공). 변형: A(디밍+비네팅) · B(블러) · C(헤더밴드+순흑) · D(순흑) · E(가운데 숫자).
3. **이미지 재사용:** 한 캐러셀에 2~3장 × 변형으로 7~10장 충당(새 이미지 최소화). 흑백 변환 금지(실사 컬러 유지).
4. **하단 스크림 강하게** — 우하단 생성 워터마크 차폐(전 슬라이드 공통).
5. **텍스트 시작 위치 고정**(스와이프 시 위아래 흔들림 금지).
6. **넘버링:** 편집용 01·02를 **핵심 메시지(제목)에 결합**. (인스타 페이지 카운터 아님.)
7. **핵심 강조:** 본문 핵심 문장은 **흰색 + 골드 밑줄**, **문단 안 인라인**(별도 줄 금지).
8. **회색 부연: 2~3문장(약 60~110자).** 너무 짧지도(1줄) 너무 길지도(5배) 않게.

### A-3. 광고법·브랜드 락 (절대) — 상세 `BRAND_LOCKS.md`
- 표기 "필수 항목 All N.D."(28종 X) · 핸들 @melanoir_official · EWG "1등급 평가"(인증 X).
- **시험성적서 공개/원본 확인 표현 금지**(정책상 비공개). → "시험 기준·데이터는 홈페이지에서".

## D. 계속 똑똑해지게 하는 법
