## A. 처리 결과

### A-1. 시험값 귀속 — ✅ 사이트 기준으로 정리
라이브 페이지가 귀속을 명확히 합니다.
- "**멜라누아의 멜라닌 색소**는 데이터로 안전을 말합니다. 피부 자극 지수 0.00, 세포 생존률 97% …" → **0.00·97% = 멜라닌 색소(원료)**
- "유해물질 불검출 / **자가품질검사** 전 항목 N.D." (제품 히어로 스펙) → **All N.D. = 완성 제품(엠보)**

따라서 사용자의 정정과 **일치**합니다: #5(0.00)·#6(97%)=‘멜라닌 소재/색소’ ✅, #7(All N.D.)=‘엠보(제품)’ ✅.
- `product-claims.json` SSoT를 이에 맞게 **정정 완료**(P-001·P-002=멜라닌 색소, P-003=제품, subject/주석 추가).
- #4(허브)는 수치를 ‘멜라누아가 시험’ 정도로만 제시(제품 단정 없음) → 그대로 무난. (원하면 0.00·97%에 ‘멜라닌 색소’ 명시 추가 가능)

### A-2. #1-01 ‘필러’ 비유 — ✅ 전략적 대체 적용
의도(=문신 염료 채택 기준을 ‘몸에 오래 남는 것’ 수준으로 격상)는 살리되, **의료기기(필러) 등가 표현은 제거**했습니다.
- 변경: "이는 마치 피부 속에 넣는 필러와도 비슷합니다" → "**몸에 오래 남는 것을 고를 때 성분부터 깐깐하게 따지듯, 색소도 같은 기준으로 봐야 합니다.**"
- 효과: ‘기준 격상’ 메시지는 유지(소비자의 기존 깐깐함을 호출), 특정 의료기기 명시는 회피 → 리스크↓.
- (대안, 원하시면 교체) ‘필러’를 굳이 쓰려면 **제품 등가가 아닌 ‘선택 기준 비유’**로만: "필러를 고를 때 성분을 따지듯…". 단 의료기기 연상 잔존 리스크 있음. → **현재는 안전한 무(無)필러 버전 적용.**

### A-3. #9 혜택·경쟁 — 캡션 ✅ / 슬라이드 ⚠️
- ‘경쟁 있음’ 확정 → 캡션의 "전원 같은 혜택·조건(한정 수량 경쟁 없음)" **삭제**, 사이트 기준으로 재작성:
  · 마감 6/30 23:59(조기 마감 가능) · 대상=전문가(인스타 포트폴리오) · **혜택: 전원 앰플 키트(3회분) / 응답 완료 시 3개월 골드 / 우수 리뷰어 3인 50만원** · 참여 3단계 · 신청 링크.
- ⚠️ **슬라이드 03 vs 사이트 불일치**(확인 요망): 슬라이드는 "골드 등급 / 우수리뷰어 50만원 / **협력 파트너십**"인데, 사이트엔 **‘협력 파트너십’ 표기 없음**, 대신 **‘무료 샘플(전원 3회분)’**이 기본 혜택으로 있음. 또 골드=‘응답 완료자’, 우수=‘3인’ 조건도 사이트에만 명시.
  → 슬라이드를 사이트와 맞출지(권장: 무료샘플 추가, ‘파트너십’은 사이트 반영 후 사용), 아니면 사이트를 업데이트할지 결정 필요. (현재 슬라이드는 오너 확인대로 유지)
