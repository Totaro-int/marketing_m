# 캐러셀 콘텐츠 설계 모델 (v1 · 골드스탠다드 #5)
**2026-06-18 · 모든 9개 캐러셀이 이 구조를 따른다.**

> 문제 진단: 제목(키워드)은 좋으나 **캐러셀을 관통하는 핵심 메시지·스토리가 없으면** 보는 사람이 "이게 뭐에 대한 내용인지" 감을 못 잡는다. 아래 4요소를 반드시 갖춘다.

## 4요소
1. **핵심 메시지(Core Message / thesis)** — 캐러셀 전체가 증명하는 한 문장. 슬라이드에 직접 안 써도, 모든 슬라이드가 이걸 향한다.
2. **후킹 커버(Cover)** — 무엇에 대한 내용인지 **즉시 감 잡히게** + 궁금증. 데이터형은 `대형 숫자(E) + 라벨 + 후킹 한 줄`. 선언형은 `헤드라인(+후킹)`.
3. **스토리 비트(본문 3~4장)** — 각 슬라이드 = `번호 + 핵심 메시지(제목)` + **회색 부연 2~3문장**. 흐름: **긴장/오해 → 척도/메커니즘 → 결과/수치 → 의미·공개**.
4. **마무리(Closing)** — 가운데 정렬·번호 없음·얇은 골드 룰. **선언 한 줄만(깔끔하게).** 링크·상세 안내·성적서 언급은 슬라이드에 넣지 않는다(약함). → **캡션(유튜브 더보기란처럼)** 에 친절한 설명 + 링크를 둔다. "→ URL" 화살표 처리 금지.

## 레이아웃 규칙
- **본문 텍스트 시작 위치는 전 슬라이드 고정** (스와이프 시 위아래로 흔들리지 않게). 제목 Y 고정, 본문은 그 아래로 흐른다.
- 워드마크 상단 고정, 데이터 커버는 가운데(E), 본문은 좌측 정렬, 마무리는 가운데.
- **핵심 강조:** 본문 3~4줄 중 핵심 문장(청크)은 **흰색 + 얇은 골드 밑줄**로 강조한다(나머지는 회색). 슬라이드당 1개. **별도 줄로 떼지 말고 문단 안에 인라인으로 자연스럽게 흐르게** 한다(줄바꿈 강제 X). 무엇을 말하려는지 즉시 보이게.

## 분량 가이드
- **캐러셀 길이: 표지 포함 7~10장** (커버 1 + 본문 5~8 + 마무리 1). 4~5장은 빈약 — 스토리 비트를 더 쪼갠다.
- 회색 부연: **슬라이드당 2~3문장(약 60~110자)**. 지금보다 2배, 단 5배는 금지.
- 번호+핵심 메시지(제목): 한 줄(자동 폭맞춤). 짧고 단정.
- 후킹 한 줄: 12~20자.

## 톤·광고법 (BRAND_LOCKS 준수)
- "안전하다/무독성" 단독·"100% 안전" 금지 → "측정 범위의 최솟값 / 자극 지수 0 / ISO 기준 / (유해물질만) 검출되지 않음". ⚠️ **용어 분리:** '검출/불검출(N.D.)'은 **유해물질 자가품질검사 전용**. ISO 10993-23(피부 자극)은 '자극 지수·측정 범위 최솟값'으로 표현(‘검출’ 금지).
- **제품 효능 주장 금지**(예: "트러블이 줄어듭니다" ❌) → "시술자가 확인해야 할 정보 / 변수 중 하나" 같은 **정보·선택** 프레임 ✅.
- ⚠️ **시험성적서 '공개/원본 확인' 표현 금지.** 멜라누아는 성적서 원본·기관명·번호를 공개하지 않는다. 사이트엔 수치·기준·데이터가 게시될 뿐이다. → "시험 기준·데이터는 홈페이지에서 확인" 으로만 안내.
- ⚠️ **레이어 분리(중요).** 브랜드 선언/여정 캐러셀(#1 등)에는 **제품 스펙 수치(0.00·97%·All N.D. = P-001~003)를 제품에 귀속하지 않는다.** "측정했다"는 접근·태도만 말하고, 실제 값은 **#5·6·7(제품 레이어, 근거 병기)** 로 넘긴다(자연스러운 퍼널). 슬로건 "가장 안전한 Black"은 **추구·정의의 선언**으로, 특정 제품의 최상급 안전 주장으로 쓰지 않는다.
- 표기: 필수 항목 All N.D. · EWG "1등급 평가"=성분(멜라닌) 분류·NGO 자체평가(제품 귀속·본문 노출 금지) · 수치 근거는 melanoir.co.kr.
- 종결 선언형, 의문문은 후킹/긴장 환기용으로만.

## 골드스탠다드 — 캐러셀 #5 「0.00」 전체 스크립트

**핵심 메시지:** 반영구 잉크는 피부에 수년간 남는다. 그래서 ‘순하다’는 말이 아니라 자극 반응이 얼마나 일어나는지를 숫자로 봐야 하며, 멜라누아 엠보는 그 값이 0.00 — 자극 반응이 나타나지 않은, 측정 범위의 최솟값이다.

| 슬라이드 | 제목(번호+핵심) | 회색 부연 | 비주얼 |
|---|---|---|---|
| **커버** | `0.00` · 피부 자극 지수 | 후킹: "‘순하다’는 말로는 알 수 없습니다." | E(가운데 숫자), `ink_on_skin` 디밍 |
| **01** | 잉크는 피부에 수년간 남습니다. | 반영구화장 잉크는 진피층에 들어가 수년간 머뭅니다. 그래서 ‘순하다’, ‘클린하다’는 표현만으로는 충분하지 않습니다. 정말 자극이 적은지, 우리는 무엇으로 확인할 수 있을까요? | C(헤더밴드) `ink_on_skin_2` |
| **02** | 그래서 척도로 봅니다. | ISO 10993-23은 피부 자극을 0에서부터 단계로 측정하는 국제 표준 시험입니다. 잉크가 피부에 자극 반응을 얼마나 일으키는지를, 느낌이 아니라 숫자로 환산합니다. | A(디밍) `ink_spread_on_glass` |
| **03** | 멜라누아 엠보의 결과는 0.00. | 이 시험에서 멜라누아 엠보의 자극 지수는 0.00이었습니다. 0.00은 측정 가능한 범위에서 가장 낮은 값입니다. ‘자극이 적다’는 표현이 아니라, 측정된 값 그 자체입니다. | C `ink_on_skin_3` |
| **04** | 그래서 우리는 이 값을 공개합니다. | 잉크가 오래 남는 만큼, 이 숫자는 시술자가 선택 전에 확인해야 할 정보입니다. 알고 선택하는 것과 모르고 쓰는 것은 다릅니다. 멜라누아는 이 값을 숨기지 않습니다. | A `ink_on_skin_2` |
| **마무리** | 멜라누아는 / 숫자로 말합니다. | (슬라이드엔 부연 없음 — 링크·설명은 캡션으로) | D(순흑) 중앙정렬 |

매핑 factIds: 커버·03 = P-001(0.00 / ISO 10993-23), 02 = F-004(시험 표준).
