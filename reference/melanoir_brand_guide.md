# 멜라누아(MELANOIR) 브랜드 가이드 — 에이전트 온보딩 입력
**2026-06-20 · `/sns-onboard`가 읽어 `company-profile.yaml`로 정리하는 단일 입력.** IG 카드뉴스 10종 작업에서 확정된 톤·금기·비주얼·사실을 distill.

## 1. 브랜드 한 줄 / 태그라인
- **태그라인:** 지구에서 가장 안전한 Black을 추구하다.
- **차별점(딱 하나):** 다른 색소를 섞지 않고 ‘몸이 만드는 색’ 멜라닌 단일 색소로 설계하고, 그 안전을 공인 시험 ‘숫자’로 공개한다.
- **산업:** 반영구화장(PMU) 색소 — 위생용품(위생용품 관리법 적용). ※광고법 민감 업종.

## 2. 브랜드 보이스 (tone.voiceNotes 로 들어갈 핵심)
- 의료·과학 **데이터 우선**. "안전하다/무독성" 단독 대신 **"검출되지 않았다 / 측정값 / ISO 기준"** 같은 팩트 톤.
- 종결: 기본 정중체("~합니다"). 헤드라인·클로징 등은 선언형("~다") 허용. 청유·감탄·이모지 금지.
- **형용사 절제**("압도적·최고의·기적" 류 금지) → 숫자·근거로 대체. 용어는 **"숫자"(O) / "수치"(X)**.
- **레이어 분리:** 교육(선언/멜라닌 일반)에는 제품 시험수치 귀속 금지. 제품 수치는 데이터 편에서 **근거(시험명) 병기**.
- **시험값 귀속(중요·성적서 기준):** 0.00(피부 자극, ISO 10993-23)·97%(세포 생존률, 세포독성)는 **‘멜라누아의 멜라닌 색소(원료)’** 값. All N.D.(유해물질 자가품질검사)는 **‘완성 제품(엠보)’** 값. 시험체와 표기를 반드시 일치.
- **검사 주체:** 시험·검사는 **공인기관**이 수행("직접/자체 측정" 표현 금지).
- **반영구 깊이:** "잉크가 늘 진피에 주입된다"식 단정 금지. 허용: "표피에 올리지만 일부는 진피까지 남는다 / 최소 일부라도 진피에 남으므로 소재가 중요".
- **마무리=넛지**(메타 안내·"다음 글에서" 금지). 링크·상세·마감은 슬라이드 아닌 **캡션**으로. 전환 링크는 **melanoir.co.kr/products/embo**.
- **표지=후킹이 주목적:** 질문형·통념 반박("‘순하다’를 숫자로 나타낼 수 있을까요?"), 데이터 표지엔 맥락 접두("반영구 잉크, 무엇이 없는지가 중요합니다"), 이미지 자체 후킹(주사기·멸균팩). 결론형 단정은 표지로 약함(단 함축적 아포리즘은 예외).
- **피드백 반영은 최소 변경 원칙**, 키워드 과반복 금지.

## 3. 금기어·표기 락 (banned + 표기 규칙)
- 표기 고정: **"필수 항목 All N.D.(불검출)"** — ‘28종/28-FREE/28종 불검출’ **금지**.
- **EWG**: "EWG가 성분(멜라닌)을 1등급으로 분류"(NGO 자체평가·인증 아님). "EWG 인증" 금지, **제품 귀속·본문 노출 금지**, 제품 시험값과 병렬 금지.
- "안전하다/무독성" 단독·"100% 안전" 금지 → 측정값/ISO/검출되지 않음.
- **의료기기·의약품 효능 표방 금지**(위생용품 관리법). **‘필러’ 등 의료기기 비유로 제품을 등가화 금지**(기준 격상은 ‘성분을 깐깐히 따진다’식 행동 비유로).
- 시험성적서 ‘원본/기관명/번호 공개’ 표현 금지 → "시험 기준·데이터는 홈페이지(melanoir.co.kr)에".
- 공식 핸들 **@melanoir_official**.
- (자동 차단) 과대광고 7패턴(치료 단정·의학적 효능·효과 보장·부작용 없음·100% 안전·최고/유일/완벽/기적/영구·즉각 효과)은 brand-guardian이 자동 block.

## 4. 비주얼 / 이미지 스타일 (imageStyle/visual)
- **실사 다크 무드 + 골드 포인트**, 흑백 변환 금지(실사 컬러 유지). 워드마크 상단.
- **디밍 그라데이션:** 상단 선명 → 절반쯤부터 충분히 어둡게 → 하단 넓고 완만하게(급격 전환 금지). **하단 워터마크는 어떤 경우에도 가려지게**(전 폭 차폐, 표지 포함).
- **이미지 반복 최소화:** 캐러셀당 같은 이미지 최대 2회, 2회째는 확대/초점이동/반전 변형. **임팩트 강한 컷(제품·니들)은 1회.** 표지 이미지는 본문 비재사용.
- **표지 이미지**는 그리드 선택 이미지 기준(토픽↔이미지 매핑). **생성형 티 나는 컷은 표지로 쓰지 않음**(본문만). **의료가운 컷 사용 금지**(병원 오인).
- 상세 이미지 분류·사용 빈도는 동봉 `IMAGE_STOCK.md` 참조.

## 5. 타깃·통점·고객 언어 (targetAudience)
- **주 타깃:** 반영구 시술 전문가(아티스트)·학원. (확장: 일반 소비자 교육)
- **결정적 순간/통점:** ‘순하다·클린하다·자연 유래’ 같은 **형용사로는 안전을 판단할 수 없다**는 불신. 고객에게 **소재 근거를 설명**해야 하는 부담. 잉크 **개봉·재사용 오염**과 위생 우려. 색소가 피부에 **수년간 남는다**는 점.
- **고객 언어(dailyVocab):** 엠보·헤어스트로크·콤보브로우, 발색, 정착, 리터칭, 딥/점도, 잔흔, 시술자.
